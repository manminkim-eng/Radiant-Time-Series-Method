📋 RTS법 부하계산 작업 순서
STEP 1 · 프로젝트 개요 입력
건물명, 용도, 위치(지역), 설계자, 계산일자, RTS Zone 번호, 안전율(SF)을 입력합니다.
※ RTS Zone은 ASHRAE 2009 Table 18 기준 (한국 서울 기준 권장 Zone 8)
STEP 2 · 기상 데이터 설정
지역 선택 시 여름 설계 DB/WB·일교차·겨울 설계 외기온이 자동 입력됩니다.
실내 설계조건(냉방 26°C·난방 20°C)을 확인하고 필요 시 수정합니다.
STEP 3 · 실별 데이터 입력
건물 용도 선택 → 실 구성 자동 편성 → 각 실의 면적·천장고·설계온도·외피(벽/창) 입력.
사용인원, 조명밀도(LPD), 기기밀도(EPD), 침기량(ACH) 입력.
STEP 3.5 · 열관류율(U값) 검토 연계
상단 🧱 열관류율 검토 버튼 → 별도 앱에서 외벽/지붕/창호 구성 입력 → U값 자동 전송.
건축물 에너지절약설계기준 지역별 기준값 초과 여부 자동 판정.
STEP 4 · 외부 열취득 입력
외벽·지붕·창호 면적, 방위각, U값, 흡수율(α), SHGC 입력.
Sol-Air 온도 테이블이 자동 생성됩니다.
STEP 5 · 내부 열취득 확인
인체·조명·기기 발열량이 실별 데이터에서 자동 계산됩니다. 필요 시 직접 수정 가능.
STEP 6 · CTS/RTS 계수 확인
선택한 Zone에 대한 24시간 CTS(전도 시계열) 및 RTS(복사 시계열) 계수 테이블이 표시됩니다.
STEP 7 · 🔄 전체 계산 실행
상단 🔄 전체 계산 버튼 클릭 → 모든 부하 자동 계산.
계산 결과 탭에서 피크 냉·난방 부하(W, kW, RT, kcal/h) 확인.
STEP 8 · AHU/FCU 선정
계산된 부하를 기반으로 공조기·팬코일 사양 입력 및 풍량 산출.
STEP 9 · 저장 및 출력
💾 파일 저장: 현재 입력 데이터를 .rts 파일로 저장 (나중에 불러오기 가능).
🖨️ A4 전체 출력: 브라우저 인쇄 → PDF 저장으로 계산서 제출용 문서 생성.
🌡️ Sol-Air 온도 계산식
t_sa = t_o + (α × E_t) / h_o − (ε × ΔR) / h_o
t_sa : Sol-Air 온도 [°C] t_o : 시각별 외기온 [°C] α : 외표면 태양 흡수율 (외벽 0.60~0.90, 지붕 0.85~0.95) E_t : 경사면 전일사량 [W/m²] h_o : 외표면 열전달계수 = 17.0 W/m²K (ASHRAE 기준) ε : 표면 반사율 보정계수 (수평 지붕 = 1, 수직 벽 = 0) ΔR : 장파복사 보정 = 63 W/m² (표준)
Sol-Air 온도는 외기온에 일사에 의한 등가 온도상승과 장파복사 손실을 합산한 개념적 온도입니다. CTS 계산의 입력값으로 사용되며, 방위·경사에 따라 시각별로 24개 값이 계산됩니다.
📌 외기온의 시각별 변화: t_o(n) = t_o,avg + (DR/2) × cos[15(n−15)]
DR = 설계 일교차 (한국 지역 일반 9~14°C 범위)
☀️ 방위별 일사량 계산
E_t = E_b × cos(θ) + E_d + E_r
E_b : 직달 일사량 [W/m²] θ : 입사각 (표면 법선과 태양 방향의 각도) E_d : 확산 일사량 [W/m²] E_r : 반사 일사량 (지면 반사율 ρ = 0.2 표준)
📊 CTS 전도 시계열법 (Conduction Time Series)
q_cond(n) = UA × Σ [c_j × (t_sa(n-j) − t_i)] (j = 0 ~ 23)
q_cond(n) : n시의 전도 열취득 [W] U : 열관류율 [W/m²K] A : 부재 면적 [m²] c_j : j시간 전 CTS 계수 (Zone별 ASHRAE Table 16) t_sa(n-j) : (n-j)시의 Sol-Air 온도 [°C] t_i : 실내 설계온도 [°C]
CTS는 ASHRAE 2009 Fundamentals Chapter 18에 정의된 24개 계수로 구성되며, 벽체/지붕의 열용량에 따른 시간 지연(thermal lag) 효과를 반영합니다. Σc_j = 1.0 이 되어야 정확한 계수 집합입니다.
📌 Zone 선택 기준: 경량(외단열/금속) → Zone 1~4, 중량(콘크리트+단열) → Zone 6~10
한국 RC조 표준 외단열 건물 → Zone 8 권장 (ASHRAE Table 16)
📈 RTS 복사 시계열법 (Radiant Time Series)
Q_cool(n) = Q_conv(n) + Σ [r_j × Q_rad(n-j)] (j = 0 ~ 23)
Q_cool(n) : n시의 냉방 부하 [W] Q_conv(n) : 대류 성분 (즉시 부하) r_j : j시간 전 RTS 계수 (Zone별 Solar/Nonsolar 구분) Q_rad(n-j): 복사 열취득 (일사 → Solar RTS, 기타 → Nonsolar RTS)
■ 열취득 분리 비율 (ASHRAE 기준) 인체 : 대류 50% / 복사 50% 조명(형광): 대류 40% / 복사 60% (LED: 대류 60% / 복사 40%) 기기 : 대류 70% / 복사 30% 외벽전도 : 대류 46% / 복사 54% 창호일사 : 대류 37% / 복사 63%
RTS 계수는 Solar(일사관련) 와 Nonsolar(그 외) 로 구분되며, Zone에 따라 달라집니다. 복사 열취득은 실내 표면(바닥·천장·벽)에 흡수된 후 시간이 지남에 따라 대류로 전환됩니다.
📌 피크 냉방 부하 = 24시간 중 Q_cool(n) 최대값 × 안전율(SF)
일반적으로 SF = 1.05 ~ 1.15 적용 (본 프로그램 기본값 1.10)
🪟 창호 열취득 계산식
■ 태양 일사 부하 q_solar = A × SHGC × E_t × IAC
■ 전도 부하 q_cond = U × A × (t_o − t_i)
SHGC : 태양열 취득계수 (Solar Heat Gain Coefficient, 0.2~0.7) IAC : 내부 차양 계수 (Internal Attenuation Coefficient, 일반 1.0) SC : 차폐계수 (Shading Coefficient) = SHGC / 0.87
📌 창호 U값은 프레임 포함 전체 창호 열관류율입니다.
SHGC가 낮을수록 일사 차단 성능이 우수합니다 (로이유리 SHGC ≈ 0.25~0.40).
⚠️ 외부 차양(오버행·수직핀) 적용 시 DF(방위 보정계수) 값을 조정하세요.
DF 기본값 = 1.0 (차양 없음). 외부 수평 차양 적용 시 ≈ 0.60~0.85.
💡 내부 발열 계산식
■ 인체 발열 (현열 기준) q_ppl = N × q_s × BF N : 재실 인원수 = 면적 × 인원밀도 [인/m²] q_s : 활동도별 1인당 현열 발열 [W/인] (사무: 65W, 경작업: 75W) BF : 재실 동시이용률 (보통 0.75~1.0)
■ 조명 발열 q_light = LPD × A × F_ul LPD : 조명 전력밀도 [W/m²] (사무 10~15, 상업 15~25) F_ul : 조명기구 이용률 (일반 1.0, 매입 0.85)
■ 기기 발열 q_equip = EPD × A × F_use EPD : 기기 전력밀도 [W/m²] (사무 10~20, 전산실 50~200)
📌 잠열 = 현열 × (잠열비/현열비). 사무작업 시 현열/잠열 = 65/55 W/인 (총 120W).
💨 침기·환기 부하
q_infil = 0.33 × n × V × Δt [W]
0.33 = ρ×c_p / 3600 ≈ 1.2 × 1005 / 3600 n : 침기 횟수 [ACH] (일반 0.05~0.15, 외주부 0.15~0.30) V : 실 체적 [m³] Δt : (외기온 − 실내온도) [K]
⚠️ 입력 주의사항
🚫 구동 시 저장파일 자동 생성 문제 수정 완료
이전 버전에서는 앱 로드 시 자동으로 .rts 저장파일이 생성되는 버그가 있었습니다.
현재 버전에서는 사용자가 💾 저장 버튼을 직접 클릭할 때만 파일이 저장됩니다.
📌 면적 입력: 실 면적은 순면적(벽체 내측 기준) 적용 권장. 창호 면적은 유리+프레임 포함 전체 창호 면적.
📌 방위각 입력: 남향=180°, 동향=90°, 서향=270°, 북향=0°(또는 360°). 정확한 방위 입력이 일사 계산 정확도에 직접 영향.
📌 U값(열관류율): 🧱 열관류율 검토 앱에서 정확한 구성 입력 후 자동 연계 권장. 임의 추정값 사용 시 오차 발생.
📌 설계 외기조건: 한국의 경우 건축물 에너지절약설계기준 별표7 (냉방기 위험률 2.5% 또는 연간 1%) 기준을 우선 적용하며, 추가 정밀 해석 시 ASHRAE Climatic Design Data 병용.
⚠️ RTS Zone 선택: 잘못된 Zone 선택은 전체 부하에 10~30% 오차 유발. 건물 외피 구성에 맞는 Zone 선택 필수.
⚠️ 안전율(SF): 과도한 안전율 적용(1.3 이상)은 과설계 원인. 일반 공조: 1.05~1.15, 클린룸/병원: 1.10~1.20 권장.
⚠️ 인접 실 조건: 비냉방 공간에 접한 벽·바닥은 간접 노출(U값 기준 완화)로 처리. 직접 외기 노출과 구분 필수.
📌 출력(PDF): 크롬 브라우저에서 '배경 그래픽 포함' 체크 후 PDF 저장 시 서식이 정확하게 출력됩니다.
